Exercise Tracker


Note: Freecodecamp's testing suite requires that the Add Workout form take a numeric ID generated by the database, when I would prefer to use the user-entered username. Also, please be aware that this app is not intended to have any other views of users or previously added workouts.
API Reference

GET users's exercise log: GET /api/exercise/log?{userId}[&from][&to][&limit]

{ } = required, [ ] = optional

from, to = dates (yyyy-mm-dd); limit = number
